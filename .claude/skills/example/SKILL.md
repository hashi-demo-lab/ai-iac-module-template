# Example Skill

This is a minimal valid skill that demonstrates the expected structure for skills in this hub. Use it as a starting point when authoring new skills.

## Usage

```
/example [arguments]
```

## What This Skill Does

When invoked, this skill simply confirms it was loaded successfully and echoes back any arguments provided via `$ARGUMENTS`.

## Instructions

1. Confirm to the user that the example skill was invoked
2. Echo back any arguments: `$ARGUMENTS`
3. Point the user to `reference/skills-authoring-guide.md` for the full authoring spec

## Notes

- This skill exists for testing and demonstration purposes
- It can be safely removed or replaced once real skills are added
- See `{baseDir}/../../reference/skills-authoring-guide.md` for the full authoring guide
